# PF2E RU Sheet Export

Модуль для Foundry VTT, который экспортирует лист персонажа Pathfinder 2e в русскоязычный PDF-шаблон.

## Основано На

- Модуль основан на проекте: https://github.com/gioppoluca/sheet-export
- Используемый русскоязычный шаблон листа взят с: https://pf2.ru/files

## Поддержка

- Система: `pf2e`
- Тип листа: `character` (PC)
- Маппинг: `RU/latest`

## Использование

1. Откройте контекстное меню персонажа в директории акторов.
2. Нажмите **Sheet Export**.
3. Нажмите **Apply**.
4. Нажмите кнопку скачивания PDF.

## Атрибуция шаблона

Русскоязычный PDF-шаблон листа персонажа взят с:
